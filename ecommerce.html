E-Commerce Store

visit site
  • This is an e-commerce store for clothing. It was built with React. It uses Firebase to store the information about clothes and allow users to create an account and login
  • Each product has its own page where the user can see a photos, a description, and the materials, choose a size, and add to cart
  • I used a Stripe API to allow users to make payments
  • There is a search function, so users can find products quicker. It looks for exact matches within a product's name, description, or material. It also ranks the matching products, depending on how many matches were found and whether those matches were in the name or not
